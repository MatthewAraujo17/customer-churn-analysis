# Customer Churn Analysis & Dashboard

## 📊 Project Overview
This project analyzes customer churn behavior and identifies key drivers of customer attrition. It includes an interactive Excel dashboard and a predictive modeling approach to segment customers by churn risk.

## 🎯 Objectives
- Identify customers likely to churn
- Analyze key factors influencing churn
- Provide actionable retention strategies

## 🧰 Tools Used
- Excel (Dashboard, Pivot Tables, Slicers)
- Data Analysis & Feature Engineering

## 📈 Key Insights
- Inactive customers have significantly higher churn rates
- Customers with only one product are most likely to churn
- High-risk customers represent a disproportionate share of churn

## 📊 Dashboard Features
- KPI summary (churn rate, total customers, risk segments)
- Churn analysis by country, age group, and activity
- Interactive slicers for filtering data
- High-risk customer identification

## 💡 Business Recommendations
- Target inactive users with engagement campaigns
- Encourage multi-product adoption
- Focus retention efforts on high-risk customers
